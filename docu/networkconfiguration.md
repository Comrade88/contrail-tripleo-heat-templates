# Advanced network Configuration

Contrail supports the following networking modes:    
1. vrouter kernel mode:
   1. standard
   2. vlan
   3. bond
   4. bond + vlan

2. vrouter dpdk mode:
   1. standard
   2. vlan
   3. bond
   4. bond + vlan

3. vrouter sriov mode:
   1. kernel mode:
      1. standard
      2. vlan
      3. bond
      4. bond + vlan
   2. dpdk mode:
      1. standard
      2. vlan
      3. bond
      4. bond + vlan
